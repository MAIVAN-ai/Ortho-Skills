# Contributing

We welcome contributions from orthopaedic surgeons, residents, radiologists, rehabilitation specialists, clinical researchers, registry experts, medical AI researchers, and software engineers.

## Pull request checklist

- [ ] No patient-identifying data
- [ ] Human-in-the-loop requirement included
- [ ] Validation level stated
- [ ] Safety constraints included
- [ ] Output schema included
- [ ] External sources linked, not copied
- [ ] No unsupported claim of clinical validation
