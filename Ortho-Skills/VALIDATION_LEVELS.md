# Validation Levels

| Level | Name | Meaning |
|---|---|---|
| V0 | Conceptual / unvalidated | Draft workflow or prompt. Not tested for clinical reliability. |
| V1 | Expert-reviewed | Reviewed by relevant clinical or technical experts, but not formally validated. |
| V2 | Retrospective case-tested | Tested on retrospective cases with known reference labels or expert judgement. |
| V3 | Multi-site retrospective validation | Tested across institutions, imaging sources, and case mixes. |
| V4 | Prospective silent-mode validation | Run prospectively without influencing care. |
| V5 | Clinical workflow validation | Validated in a controlled clinical workflow. |
| V6 | Regulatory-grade context | Used within an appropriately certified and governed system. |
